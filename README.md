# ✦ NexusAI — Hub IA Tout-en-Un

Application Android premium en Next.js avec accès à tous les meilleurs modèles IA.

## Providers & Modèles

| Provider | Modèles | Capacités |
|---|---|---|
| ⚡ **OpenAI** | GPT-5, GPT-4.1, o3, o4-mini, DALL·E 3 | Chat, Vision, Raisonnement, Images |
| 🧠 **Chipp AI** | Gemini 2.5 Pro, 2.0 Flash | Chat, Fichiers PDF, Images |
| 🔀 **RTM Router** | DeepSeek v3.1, Kimi K2, Llama 4, Qwen 3 | 25+ modèles open source |
| 🚀 **Groq** | Llama 3.3 70B, Kimi K2, GPT-OSS 120B | Ultra-rapide (LPU) |
| 📥 **Facebook DL** | — | Téléchargement vidéo HD/SD |

## Stack
- **Frontend** : Next.js 14 + TypeScript + Tailwind CSS
- **Mobile** : Capacitor 6 → APK Android
- **Backend** : Express.js sur Render (proxy sécurisé)
- **Build** : GitHub Actions (automatique sur push)

## Design
Style **light premium** — fond blanc/indigo clair, typographie Plus Jakarta Sans + Nunito,  
icônes officielles SVG de chaque IA, animations fluides, dark skeleton loader.
